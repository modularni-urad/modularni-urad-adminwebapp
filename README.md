# [Modularni urad](https://modularni-urad.github.io/) admin webclient

## how to build

```
npm run build
```

## how to debug

``` bash
# Install dependencies.
npm install

# Serve with automatic reloading.
npm start
```

#### Update zavislosti

```
git submodule update --remote --merge
```

#### init zavislosti

```
git submodule init
```
